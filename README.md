# 🧬 Malaria Detection Using Deep Learning

## 📌 Project Overview

Malaria is a life-threatening disease caused by parasites transmitted through infected mosquitoes. Early detection of malaria is essential for proper treatment and prevention.

This project presents an **AI-based malaria detection system** using **Deep Learning and Computer Vision** to automatically classify microscopic blood cell images as:

* **Parasitized** (Malaria Infected)
* **Uninfected** (Healthy Cell)

The system uses a trained **Convolutional Neural Network (CNN)** model and a **Flask web application** to allow users to upload blood cell images and receive a diagnostic prediction.

---

## 🎯 Objectives

* Detect malaria parasites in microscopic blood cell images.
* Use **Deep Learning (CNN)** for image classification.
* Build a **web interface** for easy testing and demonstration.
* Provide **confidence level and diagnostic report** for predictions.

---

## 🧠 Technologies Used

* **Python**
* **TensorFlow / Keras**
* **OpenCV**
* **Flask**
* **NumPy**
* **HTML / CSS**
* **Matplotlib / Seaborn**

---

## ⚙️ How to Run the Project

### 1️⃣ Clone the Repository

```
git clone https://github.com/navjyotikanwar85-pixel/Malaria-Detection-using-Deep-Learning.git
```

### 2️⃣ Install Dependencies

```
pip install tensorflow flask opencv-python numpy pillow matplotlib seaborn scikit-learn
```

### 3️⃣ Run the Application

```
python app.py
```

### 4️⃣ Open the Web Interface

```
http://127.0.0.1:5001
```

Upload a blood cell image and the system will generate a **malaria detection report**.

---

## 🧪 Model Details

| Feature    | Description                        |
| ---------- | ---------------------------------- |
| Model Type | Convolutional Neural Network (CNN) |
| Input Size | 64 × 64 RGB Images                 |
| Classes    | Parasitized / Uninfected           |
| Framework  | TensorFlow / Keras                 |

---

## 📊 Output Example

The system generates a **diagnostic report including:**

* Prediction Result (Parasitized / Uninfected)
* Confidence Level
* Model Information
* Processing Time
* Image Quality
* Timestamp

Example:

```
Diagnostic Report

UNINFECTED
No malaria parasite detected

Confidence Level: 98.77%

Analysis Details
Model: CustomCNN v2.0
Processing Time: 0.03 seconds
Image Quality: 640 x 640 px
Timestamp: 2026
```
